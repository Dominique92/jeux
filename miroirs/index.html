Le but est de détecter au moyen de lasers quelles cases contiennent des miroirs.

Cliquez sur les ✴ pour allumer les lasers.

Cliquez sur les cases pour marquer vos suppositions de miroirs.

Double cliquez sur les cases qui ne contiennent pas de miroir pour en vérifier le contenu.

Si vous double cliquez sur une case qui contient un miroir, vous avez perdu !

Vous avez gagné quand vous avez marqué tous les emplacements de miroirs.
